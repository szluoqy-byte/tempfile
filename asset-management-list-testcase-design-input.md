# 资产管理列表测试用例设计输入

## 1. 需求信息

| 字段 | 内容 |
|---|---|
| 需求 ID | REQ-ASSET-LIST-001 |
| 需求名称 | 资产管理列表 |
| 需求描述 | 在资产管理页面新增“资产列表”页签，支持用户按资产名称模糊搜索、按资产类型筛选并查询资产数据。页面下方以表格展示资产 ID、资产名称、类型、创建者、各类关联数量、被浏览次数、创建时间、删除时间、最后一次浏览时间，并支持分页浏览；页面文案和样式需符合产品设计要求。 |
| 本次不覆盖内容 | 不覆盖资产新增、编辑、删除和后台资产类型配置；不覆盖新增业务数据生产、数据结构变更、新增服务、安装部署、升级补丁、硬件成本、License、威胁建模、安全/隐私专项、自动化脚本、生产环境压测、内部设计文档和发布说明。 |
